## 委託流程

```
1. 📋 填寫委託表單
2. 💬 確認需求 & 報價
3. 💰 支付訂金（50%）
4. 📝 草稿確認
5. ✏️ 線稿確認
6. 🎨 上色 & 完稿
7. 💰 支付尾款（50%）
8. 📦 交付成品
```

---

## 委託須知

### ✅ 可以接的
- 原創角色（OC）
- 同人角色（Fanart）
- VTuber 立繪 / 表情差分
- 個人頭像 / SNS 用圖

### ❌ 不接的
- 仇恨、歧視相關內容
- 未經授權的商業用途
- 急件（少於 7 天）需另議

### ⚠️ 注意事項
- 完成品的著作權歸繪師所有，委託方擁有使用權
- 商用需求請走 [💼 商業合作](../business) 頁面
- R18 需求請走 [🔞 R18 委託](../r18) 頁面

## 價目表（同人 / 個人委託）

> 以下為起始價格，實際報價依複雜度調整。

| 項目 | 價格（NT$） |
|------|------------|
| 頭像 / 表情 | $500 起 |
| 半身圖 | $1,500 起 |
| 全身圖 | $3,000 起 |
| 場景插圖 | $5,000 起 |
| 角色設計（含設定） | $8,000 起 |
| LINE 貼圖組（8 張） | $4,000 起 |

---

## 加價項目

| 項目 | 說明 |
|------|------|
| 複雜背景 | 依場景複雜度另計 |
| 多人（每多一人） | 依角色數量另計 |
| 急件（7 天內） | 加收 ×1.3~2.0 |
| R18 | 加收 ×1.5~2.0 |
| 商用授權 | 請洽 [💼 商業合作](../business) |
